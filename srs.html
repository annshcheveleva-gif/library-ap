Technical Specification (SRS)

1. Introduction

Library API is a RESTful service for managing books and authors.

2. Purpose

The system allows users to manage library resources via API.

3. Functional Requirements

  • Get all books
  • Get book by ID
  • Create book
  • Delete book
  • Get authors list

4. Non-Functional Requirements

  • REST architecture
  • JSON format
  • OpenAPI 3.0 specification

5. API Endpoints

  • GET /books
  • POST /books
  • GET /books/{id}
  • DELETE /books/{id}
  • GET /authors

6. Technology Stack

  • OpenAPI 3.0
  • Python SDK
  • GitHub Pages
